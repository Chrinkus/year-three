Code Tag Examples

Inline code looks like: return new Promise();.

A block looks like:

const canvas = document.getElementById("viewport"), ctx = canvas.getContext("2d"); ctx.fillRect(10, 10, 100, 100);

Success? Yes. Appying a class with display and whitespace rules works.

function add(a, b) { return a + b; }

Individual code blocks for each line are terrible.

[1, 2, 3].map(x => x * 2) .filter(y => y % 2 == 0);

The whitespace rule is affected by auto-indent
